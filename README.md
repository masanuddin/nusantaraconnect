**NusantaraConnect** is a full-stack web platform that connects **Indonesian cultural performers** with **event organizers** through a centralized, digital booking system.  
This project demonstrates real-world problem solving, scalable backend architecture, and practical AI integration.

> *A full-stack Laravel project integrating AI to support cultural digitalization in Indonesia.*

---

## Why This Project Matters

- Cultural performers often rely on **manual, network-based bookings**
- Limited **digital visibility** reduces access to opportunities
- Event organizers struggle to find **reliable and relevant performers efficiently**
- This gap threatens both **economic sustainability** and **cultural preservation**

---

## What NusantaraConnect Delivers

NusantaraConnect provides:
- Centralized performer discovery & professional profiling  
- Efficient talent matching based on event requirements  
- Streamlined booking workflow  
- AI-assisted content generation and recommendations  

---

## Tech Stack Highlights

- **Backend:** Laravel (RESTful architecture)
- **Frontend:** Blade + Tailwind CSS
- **Database:** MySQL
- **AI Integration:** OpenAI API (ChatGPT)
- **Version Control:** Git & GitHub
- **Development Methodology:** Agile, sprint-based workflow

---

- Enables **fair access to opportunities** for cultural performers  
- Helps event organizers source talent **faster and more reliably**  
- Supports the **digitalization and sustainability of Indonesian cultural heritage**
